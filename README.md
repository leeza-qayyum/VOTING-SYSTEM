# Online Voting System

## Description
A console-based voting platform built with C++ applying Object-Oriented Programming principles. Supports role-based access for voters and administrators.

## Technologies Used
- C++
- OOP (Encapsulation, Inheritance, Polymorphism)

## How to Run
1. Compile the C++ files
2. Run the program
3. Login as Admin or Voter

## Features
- Role-based authentication
- Real-time vote tallying
- Instant result declaration
